# Daily Digest

Daily curated feed of AI, tech, and science news.

## Feed

**RSS:** https://raw.githubusercontent.com/shaunporwal/daily-digest/main/feed.xml

Subscribe in NetNewsWire or any RSS reader.

## Structure

- `entries/` — JSON files with each day's digest
- `feed.xml` — Generated RSS 2.0 feed
- `generate-feed.py` — Regenerates RSS from entries
- `publish-digest.py` — Creates new entry + commits + pushes

## Updates

Published twice daily:
- **9 AM ET** — Morning briefing
- **5 PM ET** — Evening update

Topics covered:
- 🤖 AI Models & Innovations
- 🧬 Science & Biotech
- 💻 Technology Trends
- 🌍 Global News
